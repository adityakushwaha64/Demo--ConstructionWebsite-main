# 🏗️ Construction Website

A modern and fully responsive Construction Website developed using **HTML5, CSS3, and JavaScript**. The website is designed for construction companies to showcase their services, completed projects, team members, company information, and contact details with a clean and professional user interface.

---

## 📖 Overview

This project is a front-end construction company website created to provide a modern online presence for construction businesses. It features responsive layouts, attractive UI components, and smooth navigation, making it suitable for desktops, tablets, and mobile devices.

## 💻 How to Run the Project

### Method 1

Clone the repository

```bash
git clone https://github.com/your-username/ConstructionWebsite-main.git
```

Open the project folder.

Double-click **index.html**

OR

---

### Method 2 (Recommended)

1. Open the project in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click on **index.html**.
4. Click **Open with Live Server**.

The website will open automatically in your default browser.
